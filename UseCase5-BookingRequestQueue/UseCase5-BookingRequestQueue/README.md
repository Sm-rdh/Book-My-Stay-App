# Use Case 5: Booking Request Queue (FIFO)

## 📌 Description
Handles booking requests using a queue to ensure fairness.

## 🎯 Goal
Process booking requests in First-Come-First-Served order.

## 🧩 Concepts Used
- Queue (LinkedList)
- FIFO
- Fair request handling

## 🚀 How to Run
javac *.java  
java UseCase5BookingRequestQueue

## 🧪 Output
Booking Request Queue  
Processing booking for Guest: Abhi, Room Type: Single  
Processing booking for Guest: Subha, Room Type: Double  
Processing booking for Guest: Vannathi, Room Type: Suite
