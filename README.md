# 🏨 Book My Stay App

A comprehensive **Hotel Booking Management System** built using **Core Java and Data Structures**, designed to simulate real-world booking workflows and system design principles.

---

## 📌 Overview

This project demonstrates how fundamental programming concepts and data structures can be applied to build a scalable and maintainable booking system.

Each use case incrementally introduces a real-world concept such as:

* Fair request handling
* Inventory consistency
* Booking lifecycle management
* Concurrency handling
* Data persistence

---

## 🚀 Key Features

* 🧾 Booking request queue (FIFO)
* 🏨 Room allocation with inventory control
* ➕ Add-on service integration
* 📊 Booking history & reporting
* ⚠️ Error handling & validation
* 🔁 Booking cancellation with rollback
* 🧵 Concurrent booking simulation (thread safety)
* 💾 Data persistence & recovery

---

## 🧠 Concepts Covered

### 🔹 Core Java

* Classes & Objects
* Inheritance & Abstraction
* Polymorphism & Encapsulation
* Exception Handling
* File I/O & Serialization
* Multithreading

### 🔹 Data Structures

* Queue (FIFO booking requests)
* Stack (rollback operations)
* HashMap (inventory management)
* List (booking history)
* Set (unique room allocation)

### 🔹 System Design Principles

* Separation of concerns
* Single source of truth
* Fail-fast design
* Thread safety
* State management & recovery
